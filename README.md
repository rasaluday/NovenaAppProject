MediPlus App – Django Hospital Management System

📌 Overview
MediPlus is a web-based Hospital Management System built using Django.
It allows patients to book appointments online and helps admins manage hospital operations efficiently.

🚀 Features:

👤 Patient Side
User Registration & Login
Book Appointments with Doctors
View Appointment Details
Secure Authentication

🛠️ Admin Side
Admin Dashboard
Manage Patients
Manage Doctors
Manage Appointments

🛠️ Tech Stack
Backend: Python, Django
Frontend: HTML, CSS, Bootstrap, JavaScript
Database: MySQL / SQLite
Tools: Git, VS Code, Postman

Setup Instructions:
# Clone repository
git clone https:https://github.com/rasaluday/mediplus.git

# Move to project folder
cd mediplus

# Create virtual environment
python -m venv venv

# Activate environment
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run migrations
python manage.py migrate

# Start server
python manage.py runserver


Key Functionalities:
1.Authentication system (Login/Register)
2.Appointment booking system
3.CRUD operations (Patients, Doctors, Appointments)
